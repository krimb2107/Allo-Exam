Premiérement il faut que vous téléchargez Pyton2.7 et le mettre dans le path.

Deuxiément il faut que vous changer les adresses ip dans les fichiers suivant par votre propre adresse IP:  SERVER => DISPLAY.HTML => line : 19.
                          CLIENT => APP.JS       => line : 06.

Les étapes à suivre:
1/ Ouvrir le dossier dans VS Code.
2/ Ouvrir votre terminal bash.
3/ Taper la commande suivante: 'cd server' .
4/ Taper la commande suivante: 'npm install' .
5/ Taper la commande suivante: 'node index.js' => à ce niveau là, la partie seveur sera lancer, et donc le boutton de contrôle dans l'application web sera accessible.

6/ Ouvrir un 2ème terminal bash.
7/ Taper la commande suivante: 'cd client'.
8/ Taper la commande suivante: 'npm install'.
9/ Taper la commande suivante: 'npm start' => à ce niveau là, le logiciel desktop va se lancer sur votre bureau et c'est à vous de découvrir!!

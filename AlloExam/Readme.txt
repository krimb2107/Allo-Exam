Premiérement il faut que vous disposez d'un serveur local & un éditeur de code de préférence WampServer et VS Code.

Deuxiément il faut que vous changer l'adresses ip dans le fichier suivant par votre propre adresse IP si vous être le professeur: 
  CLIENT => SRC => COMPONENTS => UI => CALLPAGEFOOTER => CALLPAGEFOOTER.JS => line : 17.

Les étapes à suivre:
1/ Ouvrir le dossier dans VS Code.
2/ Ouvrir votre terminal bash.
3/ Taper la commande suivante: 'npm install' .
4/ Taper la commande suivante: 'cd client' .
5/ Taper la commande suivante: 'npm install .
6/ Taper la commande suivante: 'npm start' .

juste après le serveur va se lancer et l'application va s'ouvrir automatiquement dans votre navigateur par défaut.

Et bonne découverte!!
